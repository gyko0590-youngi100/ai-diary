기본 환경 설정

STEP 01 완료

Homebrew 설치

맥용 패키지 매니저 설치. macOS 12.4라서 경고가 떴지만 M1 칩이면 문제없이 진행 가능.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
STEP 02 에러

brew: command not found

Homebrew 설치 후 brew 명령어를 인식하지 못하는 문제 발생.

zsh: command not found: brew

💡 해결 방법

PATH 환경변수에 Homebrew 경로 추가

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
🐳

Docker 설치 시도 (실패)

STEP 03 실패

Docker Desktop 호환성 문제

처음에는 Open WebUI를 Docker로 설치하려 했으나, Docker Desktop이 macOS Sonoma 이상만 지원해서 실패.

Error: This software does not run on macOS versions older than Sonoma.

Colima + QEMU 조합도 시도했으나 설치 시간이 너무 오래 걸려서 방향 전환.

🤖

텔레그램 봇 만들기 (첫 번째 시도)

STEP 04 완료

Python 봇 직접 제작

Docker 대신 Python으로 직접 텔레그램 봇을 만들기로 결정. BotFather로 봇 토큰 발급받고, Claude API 키 발급.

pip3 install python-telegram-bot anthropic
STEP 05 에러

API 크레딧 없음

봇은 만들었지만, Claude API 크레딧이 없어서 작동하지 않음. Claude Max 구독과 API 크레딧은 별개라는 걸 알게 됨.

Error: Your credit balance is too low to access the Anthropic API.
🦞

OpenClaw 설치

STEP 06 완료

Node.js 설치

OpenClaw 실행을 위해 Node.js 설치. Homebrew에 문제가 있어서 공식 사이트에서 직접 다운로드.

STEP 07 완료

OpenClaw 설치 및 설정

OpenClaw 설치 후 DuckDuckGo 검색, 스킬들 설정.

npx openclaw onboard
STEP 08 에러

grammy 모듈 없음

텔레그램 플러그인이 grammy 모듈을 찾지 못하는 문제 반복 발생.

💡 해결 방법

OpenClaw를 전역 설치하니 해결됨

sudo npm install -g openclaw
🔐

Claude OAuth 연결

STEP 09 핵심!

Claude CLI로 OAuth 토큰 발급

Claude Max 구독을 활용해 API 크레딧 없이 사용하는 방법 발견! Claude CLI로 OAuth 토큰을 발급받아 OpenClaw에 연결.

claude setup-token
openclaw models auth paste-token --provider anthropic
STEP 10 완료

Claude Opus 4.5 기본 모델 설정

기본 모델을 최강 모델인 Claude Opus 4.5로 설정.

openclaw config set agents.defaults.model anthropic/claude-opus-4-5
📱

텔레그램 연결

STEP 11 완료

텔레그램 봇 채널 추가

OpenClaw에 텔레그램 봇 토큰 연결.

openclaw channels add --channel telegram --token [봇토큰]
STEP 12 승인 필요

페어링 승인

텔레그램에서 처음 메시지를 보내면 "access not configured" 메시지가 나옴. 페어링 코드로 승인 필요.

openclaw pairing approve telegram [페어링코드]
