삽질의 시작

STEP 01 완료

터미널이 뭔데?

맥에서 Command + Space 누르고 "터미널" 검색.

나는 까만 화면이 나올 줄 알았는데... 흰색 창이었다. (설정에 따라 다르다고 함)

어쨌든 여기에 명령어를 입력하면 컴퓨터가 일을 한다. 무섭지만 일단 시작!

STEP 02 완료

Claude한테 물어보며 시작

"나만의 AI 비서를 만들고 싶어. 도와줘."

Claude가 친절하게 알려줬다. 먼저 Homebrew를 설치하란다.

Homebrew = 맥용 앱스토어 (개발자 버전)
개발자들은 앱스토어 대신 이걸로 프로그램을 설치한다.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
STEP 03 에러

첫 번째 에러 — "명령어를 찾을 수 없습니다"

설치는 됐는데 brew 명령어가 안 먹힘.

zsh: command not found: brew

비유하면: 앱은 깔았는데 바탕화면에 아이콘이 안 생긴 것.

💡 해결

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
💀

여기서 2시간 날림...

STEP 04 진로 변경

Claude와 싸워가며...

GPTers 웨비나에서 어깨너머로 본 내용이랑 너무 달랐다.

Docker 설치? 안 됨. Python 봇? API 크레딧 없어서 안 됨.

Claude와 싸워가며 (진짜 싸웠다) 결국 결론 내림:

"야, 그냥 OpenClaw 설치하는 것만 도와줘!"

진로 변경. OpenClaw 직접 설치하기로 했다.

🌟

드디어 희망! OpenClaw

STEP 05 완료

OpenClaw 설치

OpenClaw = 여러 메신저(텔레그램, 카톡, 슬랙 등)로 AI랑 대화할 수 있게 해주는 프로그램

npx openclaw onboard

이 한 줄로 기본 설치 완료!

근데 또 에러... grammy 모듈을 찾을 수 없습니다

💡 해결: 전역 설치로 바꿈

sudo npm install -g openclaw
🍯

핵심 꿀팁! Claude 연결

STEP 06 핵심!

OAuth 토큰으로 무료 사용

여기가 가장 중요한 부분.

Claude 구독자는 OAuth 토큰이라는 걸로 추가 비용 없이 API를 쓸 수 있다.

claude setup-token
openclaw models auth paste-token --provider anthropic

💡 비유

• 일반 API = 선불 교통카드 (충전해야 씀)
• OAuth 토큰 = 정기권 (구독하면 무제한)

나는 이미 Claude Max 구독 중이니까 정기권 쓰면 되는 거였다!

STEP 07 완료

텔레그램 연결 — 완성!

텔레그램에서 @BotFather한테 말 걸어서 봇 만들고, 받은 토큰을 연결.

openclaw channels add --channel telegram --token [내봇토큰]

처음 메시지 보내면 "승인 필요"라고 나오는데, 터미널에서 승인해주면 끝!

🐉
