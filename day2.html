🗣️ 한국어 TTS 삽질기

문제: 용에게 목소리를 줘야 해!

자기소개 영상을 만들려면 한국어 남자 목소리가 필요했다.
근데... 생각보다 어려웠다. 🥲

서비스 한국어 남자 품질 결과

Google TTS ✅ 느림 로봇 같음
macOS Yuna ❌ 여자만 보통 남자 없음
Edge TTS ✅ InJoon 좋음 🏆 채택!

💡 해결책: Microsoft Edge TTS

무료이면서 자연스러운 한국어 남자 음성 발견!

edge-tts --voice ko-KR-InJoonNeural --rate=+35% --pitch=-8Hz --text "충성!" --write-media output.mp3

rate로 속도 조절, pitch로 목소리 톤 조절 가능

🎬 용 자기소개 영상

제작 과정
  1. 이미지 생성 - Playground AI로 용 캐릭터 생성

  3. 음성 생성 - Edge TTS로 한국어 대사
  4. 합성 - ffmpeg로 영상 + 음성 결합
